# CMP Demo - Landing Page with AI Chatbot

A modern demo landing page featuring the Cosentus ClaimsManager Portal navbar and an integrated AI chatbot.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Deployment

This project is configured for easy deployment on Vercel:

```bash
vercel
```

## Features

- Responsive navbar matching ClaimsManager Portal design
- Modern landing page layout
- AI-powered chatbot integration
- Optimized for Vercel deployment
